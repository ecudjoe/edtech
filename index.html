SECTION THREE PRACTICAL ONE

Question 1: Assemble a PC

Describe the steps you would take to assemble these components into a fully functioning PC.

Answer:

  1. Install the CPU on the motherboard, securing it in place with the lever.
  2. Apply thermal paste to the top of the CPU, then attach the CPU cooler.
  3. Insert the RAM sticks into the appropriate slots on the motherboard.
  4. Mount the motherboard in the case, ensuring it is securely screwed down.
  5. Install the power supply unit in the case and connect it to the motherboard and other components as necessary.
  6. Connect the hard drive or SSD to the motherboard using a SATA cable and secure it within the case.
  7. Install any additional peripherals such as a graphics card, sound card, etc.
  8. Connect all necessary cables, including power cables, to the motherboard and peripherals.
  9. Close the case and connect the monitor, keyboard, and mouse.
  10. Power on the computer and enter the BIOS to ensure all components are detected and functioning properly.
  11. Install the operating system and necessary drivers.

Question 2: Troubleshooting a Monitor Issue

Your computer seems to be running, but nothing is displayed on the monitor. List the steps you would take to troubleshoot this problem.

Answer:

  1. Check if the monitor is turned on and properly plugged into a power source.
  2. Ensure the video cable (HDMI, VGA, DVI, etc.) is securely connected to both the monitor and the computer.
  3. Test with a different cable or monitor to determine if the issue is with the monitor or the computer.
  4. Check for any damage to the cable and replace it if necessary.
  5. Inspect the graphics card and ensure it is properly seated in its slot on the motherboard.
  6. Restart the computer and listen for beep codes during boot, which can indicate hardware issues.
  7. If using a laptop, try connecting an external monitor to determine if the issue lies with the built-in display.

Question 3: Installing and Configuring Microsoft Office

Explain how you would install Microsoft Office on a new computer and activate it using a product key.

Answer:

  1. Visit the Microsoft Office website and purchase the software or retrieve your existing account details.
  2. Download the Office setup file from the official Microsoft website or insert the installation disc into your computer’s drive.
  3. Run the installation file and follow the on-screen instructions to install Office.
  4. During installation, enter your product key when prompted to activate your Office suite.
  5. Once the installation is complete, open any Office application like Word or Excel.
  6. Follow any additional on-screen instructions to complete activation, which might include signing into your Microsoft account.

Question 4: Setting Up Email Filters in Gmail

Explain how to set up a filter in Gmail to automatically categorize emails from "info@example.com" into a folder named "Information".

Answer:

  1. Log into your Gmail account.
  2. Click on the gear icon to open Settings, then go to ‘See all settings’.
  3. Click on the ‘Filters and Blocked Addresses’ tab and then on ‘Create a new filter’.
  4. In the From field, enter "info@example.com".
  5. Click on ‘Create filter with this search’.
  6. In the next window, check the box ‘Apply the label’ and choose ‘New label’ from the dropdown menu.
  7. Name the label "Information" and click ‘Create’.
  8. Check the box 'Also apply filter to matching conversations' if you want to categorize existing emails.
  9. Click on ‘Create filter’ to save your settings.

Question 5: Creating a Simple Cybersecurity Plan

Create a simple cybersecurity plan for a small business that includes measures for data protection, threat prevention, and incident response.

Answer:

  1. Data Protection: Implement strong encryption for storing and transmitting data. Use secure, multi-factor authentication methods for accessing sensitive information.
  2. Threat Prevention: Install and maintain up-to-date antivirus software on all devices. Use firewalls to protect against unauthorized access to the network.
  3. Incident Response: Develop a formal incident response plan that includes identification of an incident, containment strategies, eradication of the threat, and recovery steps. Train all employees on the procedures to follow in case of a cybersecurity breach.

SECTION FOUR PRACTICAL TWO

Question 1: What are the key features of second generation computers?

Answer:

Second generation computers were based on transistor technology which was a significant improvement over the vacuum tubes used in first generation computers. They were smaller, faster, more reliable, and more energy-efficient than their predecessors. Examples include IBM 7094 series, CDC 1604, and UNIVAC 1108.

Question 2: How would you configure an email signature in Microsoft Outlook?

Answer:

To configure an email signature in Microsoft Outlook, go to File > Options > Mail > Signatures. Click on "New" to create a new signature and enter your desired text. You can format your signature with different fonts, sizes, colors, and include images or links. Once created, set the signature to automatically append to all outgoing messages or select specific messages manually.

Question 3: Create a document in Word with an automatic Table of Contents based on heading styles.

Answer:

Open Microsoft Word and type your document using Heading 1, Heading 2, etc., for your section titles. To insert a Table of Contents, go to the References tab, click on Table of Contents, and select a style. Word will automatically generate a Table of Contents based on the headings used in your document.

Question 4: Explain how to use VLOOKUP in Excel.

Answer:

VLOOKUP is used to search for a value in the first column of a range or table, and return a value in the same row from a specified column. To use VLOOKUP, enter =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup]) into a cell. Replace 'lookup_value' with the value you want to search for, 'table_array' with the range to search within, 'col_index_num' with the column number of the data to be returned, and 'range_lookup' with TRUE for approximate match or FALSE for an exact match.

SECTION FIVE PRACTICAL THREE

Question 1: How can you use Excel to remove duplicates from a list of entries?

Answer:

Select the range where duplicates are located. Go to the Data tab, and click on 'Remove Duplicates'. Select the columns you want to check for duplicates, and press OK. Excel will delete duplicates and keep unique records.

Question 2: Describe how to split a single cell's text into multiple columns using the Text to Columns feature.

Answer:

Select the column with text. Go to the Data tab, and click 'Text to Columns'. Choose 'Delimited' if your data is separated by characters such as commas or tabs, or 'Fixed width' if the data locations are fixed. Click Next, set your delimiters or column breaks, then click Finish.

Question 3: How would you calculate the weighted average of a set of values in Excel?

Answer:

Use the SUMPRODUCT formula along with SUM. For instance, if values are in A1:A5 and their respective weights in B1:B5, use =SUMPRODUCT(A1:A5, B1:B5) / SUM(B1:B5).

Question 4: What function would you use to count the number of sales transactions that occurred on a specific date?

Answer:

Use the COUNTIF function. If dates are in A1:A100 and the specific date to check is in B1, use =COUNTIF(A1:A100, B1).

Question 5: Explain how to create a macro that automatically formats new data as it is entered into Excel.

Answer:

To create a macro for formatting, go to the Developer tab, click Record Macro, perform the formatting actions you require, and stop recording. To make it automatic, you might need to write a simple event-handler VBA code that triggers formatting actions when data is entered into specific cells.

Question 6: How do you create a histogram in Excel to analyze the distribution of a dataset?

Answer:

Select the data, go to the Insert tab, click on the 'Insert Statistic Chart', and then select 'Histogram'. Excel will automatically create a histogram, and you can adjust the bin range and other settings under the Format Axis options.

Question 7: Describe how to set up data validation to prevent incorrect data entry in a cell that requires a specific date range.

Answer:

Select the cell(s), go to Data > Data Validation, set Criteria to 'Date' under 'Allow:', specify the 'Start date' and 'End date' under 'Data' and choose 'between'. Enter the date range, then click OK.

Question 8: What steps would you take to perform a two-way lookup using INDEX and MATCH functions?

Answer:

Use INDEX to retrieve the value at the intersection of a specific row and column. Use MATCH to find the row and column index numbers. For instance, =INDEX(A1:C100, MATCH("RowVal", A1:A100, 0), MATCH("ColVal", A1:C1, 0)) where "RowVal" and "ColVal" are the lookup values for the row and column, respectively.

Question 9: How would you automate the sending of an Excel report via email at the end of each month?

Answer:

This can be achieved by using VBA to create a macro that utilizes Microsoft Outlook to send the workbook as an attachment. Use the Workbook_BeforeClose event to trigger the email function, ensuring it checks the system date and only sends on the last day of the month.

Question 10: Explain how to connect Excel to an external data source, such as SQL Server, and import data into a spreadsheet.

Answer:

Go to Data > Get Data > From Database > From SQL Server Database. Enter the server name and database from which you want to import data. After connecting, select the tables or write a SQL query to specify the data you want, then load it into the Excel sheet.

The COUNTIF function is used for conditional counting. For example, =COUNTIF(D1:D10, ">20") counts how many cells in the range D1:D10 contain numbers greater than 20.

The IF function performs logical comparisons. Example: =IF(A1 > 10, "Over 10", "10 or less") checks if A1 is greater than 10 and returns "Over 10" if true, and "10 or less" if false.

Go to 'File' > 'Export' > 'Create PDF/XPS Document' > 'Create PDF/XPS'. Choose your options for optimization, and click 'Publish' to generate a PDF of your publication.

Question 17: How to record a slideshow with narration in PowerPoint?

Answer:

To record a slideshow with narration, go to the 'Slide Show' tab and select 'Record Slide Show'. Choose to record from the current slide or from the beginning, and use the controls to record your narration as you progress through the slides.

Question 18: How to use form controls to enter data in Access?

Answer:

Use the form wizard or design a form manually in Access to provide a user-friendly interface for data entry. Add controls like text boxes, combo boxes, and buttons to manage data input effectively in forms.

Question 19: How to link tables in Access to external data sources?

Answer:

Use the External Data tab in Access to link tables from sources such as SQL databases, Excel spreadsheets, or other Access databases. Choose 'Link to the data source by creating a linked table' and follow the prompts to establish the connection.

Question 20: How to create custom slide layouts in PowerPoint?

Answer:

In PowerPoint, go to 'View' > 'Slide Master'. Select the Slide Master and click 'Insert Layout'. Customize by adding placeholders, arranging them as desired, and using the background and theme options to design your layout. Save and close the master view to use your new layout.
